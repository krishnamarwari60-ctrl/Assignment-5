## Assignment 5 DATA ANALYSIS WITH SQL


## Description
This task evaluates student's performance in individual subject using sql.
Each subject score is checked against a minimum passing mark of 40.
A CASE statement assigns a Pass or Fail status for each subject.
The output displays the student ID, subject scores, and their results. individual subjects and determine their Pass or Fail status based on a predefined passing criterion using SQL.


##  Tasks Performed
1. Created a student score table to store subject-wise marks.
2. Assigned grades to students based on their total scores using SQL CASE statements.
3. Evaluated pass/fail status for each subject using a defined passing threshold.
4. Displayed student IDs, scores, grades, and subject-wise results for performance analysis.
